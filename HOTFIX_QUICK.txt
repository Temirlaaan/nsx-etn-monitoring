# ⚡ БЫСТРОЕ ИСПРАВЛЕНИЕ

## Проблема
Веб-интерфейс выдает ошибку 500 при открытии.

## Решение (30 секунд)

```bash
# 1. Остановить
docker compose down

# 2. Скачать новый архив и распаковать
tar -xzf etn-cert-monitor.tar.gz

# 3. Пересобрать и запустить
cd etn-cert-monitor
docker compose up -d --build

# 4. Проверить
docker compose logs -f
```

## Открыть веб
```
http://localhost:8000
```

**Теперь должно работать!** ✅

Увидите 4 ETN в таблице со статусом "Не проверялся" (это нормально, проверка по расписанию).
